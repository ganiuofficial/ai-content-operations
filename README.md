## AI Content Operations Platform

### Overview

This project was built to eliminate repetitive work in video content production while maintaining human editorial oversight.

It started as a subtitle quality assurance workflow.

After repeated use, it evolved into a reusable content operations platform capable of:

* Proofreading subtitle files
* Preserving timestamps
* Maintaining terminology consistency
* Producing audit reports
* Repurposing finished content into multiple publishing formats

The objective was simple:

> Reduce repetitive manual work while improving consistency and quality.

---

## The problem

Every video required me to manually:

* proofread subtitles
* compare edits
* verify names
* generate reports
* rewrite the same content for multiple platforms

This often consumed more than an hour per video.

The process was repetitive.

It wasn't scalable.

---

## The solution

I designed an AI-assisted workflow that:

### Subtitle QA

* Uploads an SRT file
* Protects timestamps
* Corrects grammar
* Corrects punctuation
* Preserves protected terminology
* Maintains transcript consistency
* Produces a downloadable corrected subtitle file

---

### Audit System

Automatically generates

* proofreading report
* change log
* flagged names
* failed line recovery report

Instead of asking: *"Was anything changed?"*, the workflow answers *"Exactly what changed?"*

---

### Content Repurposing

Using the cleaned transcript, it automatically creates

* LinkedIn post
* LinkedIn article
* Facebook post
* Instagram caption
* WhatsApp Channel post
* YouTube title
* YouTube description
* YouTube Community post
* X post
* TikTok script
* Website article
* Thumbnail text ideas

The tone and audience are configurable.

---

## Results

### Before

Manual subtitle QA: 90–95 minutes

---

### After

Approximately: 25 minutes

Reduction: **74%** while improving consistency and documentation.
